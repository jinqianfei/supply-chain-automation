# 飞书文件发送示例

## 场景

用户上传订单 → AI处理 → 生成Excel → **直接发送Excel给用户**

---

## OpenClaw Agent 处理流程

```python
# 处理订单
result = skill.execute(order_input=user_file)

if result["success"]:
    # 发送Excel文件给用户
    message.send(
        media=result["output_file"],        # 文件路径
        file_name=result["file_name"],        # 文件名
        target=user_id
    )
    
    # 同时发送结果摘要
    message.send(
        message=result["message"],
        target=user_id
    )
else:
    # 错误处理
    message.send(
        message=f"❌ {result['error_message']}",
        target=user_id
    )
```

## OpenClaw 发送文件的方式

### 方式1：直接发送文件路径

```python
message.send(
    media="/path/to/华鼎出库单_20260602.xlsx",
    target=user_id
)
```

### 方式2：使用filePath参数

```python
message.send(
    file_path=result["output_file"],
    target=user_id
)
```

### 方式3：读取文件后发送buffer

```python
with open(result["output_file"], "rb") as f:
    buffer = base64.b64encode(f.read()).decode()
    
message.send(
    buffer=buffer,
    filename=result["file_name"],
    target=user_id
)
```
