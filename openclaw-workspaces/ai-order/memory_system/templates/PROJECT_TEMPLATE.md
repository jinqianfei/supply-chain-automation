# 项目记忆模板

> 每个项目一个目录，结构统一，方便跨会话追溯。

```
projects/<项目名>/
├── PROJECT.md # 项目概述（必填）
├── sessions/              # 按会话组织总结
│   ├── YYYY-MM-DD-session-xxx.md
│   └── ...
├── files/                 # 生成的文件索引
│   └── INDEX.md
├── outputs/               # 产出物（报告/文档/截图）
│   └── INDEX.md
├── problems/              # 问题追踪
│   └── INDEX.md
└── skills/                # 使用的 skill 版本
    └── INDEX.md
```

---

```markdown

**最后更新：** YYYY-MM-DD
**状态：** 🟢 活跃 | 🟡 进行中 | 🔴 暂停 | ✅ 完成
**关联用户目标：** ...

3. ...
```
