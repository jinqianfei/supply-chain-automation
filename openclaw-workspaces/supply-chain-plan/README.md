# AI 获客自动化工具 v1.0

> 用 AI 自动搜索目标客户、自动生成个性化邮件、自动发送

---

## 功能介绍

### 三大核心能力

| 能力 | 说明 |
|------|------|
| **智能搜索** | 用 Tavily AI 搜索目标企业，自动提取联系方式 |
| **自动生成** | 根据你的话术模板，AI 自动生成个性化邮件 |
| **批量发送** | 支持 163/QQ 等国内邮箱，批量自动发送 |

### 工作流程

```
搜索目标企业 → 提取联系方式 → 生成个性化邮件 → 自动发送
```

## 安装配置

### 1. 配置邮件发送

编辑 `config.py`：

```python
EMAIL_CONFIG = {
    "smtp_server": "smtp.163.com",
    "smtp_port": 465,
    "email": "你的邮箱@163.com",
    "password": "邮箱授权码"  # 不是登录密码，是SMTP授权码
}
```

**获取邮箱授权码：**
- 163邮箱：设置 → POP3/SMTP/IMAP → 开启 → 获取授权码
- QQ邮箱：设置 → 账户 → POP3/SMTP服务 → 生成授权码

### 2. 配置 AI 搜索（可选）

```python
TAVILY_API_KEY = "你的Tavily API Key"  # 免费注册：app.tavily.com
```

> 不配置也可以使用，但搜索精度会下降

### 3. 配置话术模板

编辑 `config.py` 中的 `EMAIL_TEMPLATE`：

```python
EMAIL_TEMPLATE = """您好，{contact}...

{你的话术}

祝好
"""
```

## 使用方法

### 方法一：全自动模式

```bash
python3 auto_sender.py
```

会执行：
1. 搜索目标企业
2. 提取联系方式
3. 生成邮件
4. 保存到 `outputs/draft.csv`（不自动发送）

### 方法二：发送模式

```bash
python3 auto_sender.py --send
```

会执行完整流程并自动发送邮件。

### 方法三：仅发送已有名单

把你的企业名单放到 `inputs/companies.csv`，格式：

```csv
公司名称,行业,联系人,邮箱
上海某某机械,精密制造,王总,wang@example.com
```

然后运行：

```bash
python3 send_from_csv.py --file inputs/companies.csv
```

## 数据来源

### 支持的搜索渠道

| 渠道 | 联系方式获取率 |
|------|----------------|
| 百度/Google 搜索 | 中等 |
| 1688 店铺 | 需要手动提取 |
| 行业网站 | 取决于具体网站 |
| 用户自己提供 | 最高 |

### 用户自提供数据格式

推荐使用这种方式：
- 从企查查/天眼查导出企业名单
- 从1688店铺收集联系人信息
- 从展会名单获取联系方式

整理成 CSV 格式即可直接导入发送。

## 常见问题

### Q: 邮件发送失败怎么办？

检查：
1. 邮箱是否开启了 SMTP 服务
2. 授权码是否正确
3. 网络是否正常

### Q: 搜索不到企业邮箱？

这是正常的，很多小企业官网没有公开邮箱。

解决方案：
1. 使用"用户自提供数据"模式
2. 手动在1688/百度爱采购查找
3. 用企查查导出企业信息

### Q: 邮件进入垃圾箱怎么办？

1. 避免在正文中过多营销词汇
2. 使用企业邮箱而非个人邮箱
3. 控制发送频率（每天不超过50封）
